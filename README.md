# Atlas Secure Gateway - وكالة أتلانتا سند للتأمين

موقع إلكتروني لوكالة تأمين معتمدة من Atlanta Sanad يقدم خدمات التأمين للسيارات والشاحنات والشركات والسفر.

## التقنيات المستخدمة

هذا المشروع مبني باستخدام:

- **Vite** - أداة البناء السريعة
- **TypeScript** - لغة البرمجة
- **React** - مكتبة واجهة المستخدم
- **shadcn-ui** - مكونات واجهة المستخدم
- **Tailwind CSS** - إطار عمل CSS
- **React Router** - للتوجيه
- **Framer Motion** - للحركات والانتقالات

## متطلبات التشغيل

- Node.js (الإصدار 18 أو أحدث)
- npm أو yarn

## التثبيت والتشغيل

```sh
# تثبيت المكتبات المطلوبة
npm install

# تشغيل خادم التطوير
npm run dev

# بناء المشروع للإنتاج
npm run build

# معاينة البناء
npm run preview
```

## الميزات
